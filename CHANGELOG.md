## [1.0.0] - 2026-03-18

### Added

- Web Warriors, Star Turn, Plans Within Plans, Regenesis, and Inner Devil Gallery Card Themes
- Show proficiency level on card
- Announcement, profile, and credit functionality
- Reusable modals for announcements, profile, and credits

### Changed

- Hero name scales with card size
- CSS improvements

### Fixed

- Hero level is now default to 1 instead of null

## [0.3.0] - 2026-03-15

### Added

- Proficiency icons
- Toggle to show hero names
- Hero card size options
- About section in settings

### Changed

- Improved readability for customize and settings
- Scrollbar disappearing does not move the settings icon

## [0.2.0] - 2026-03-14

### Added

- Settings page
- Changes are saved in localStorage
- Import/export saved data

### Changed

- Rank colors to better match the game

### Fixed

- Unreadable ranks and levels in pop-up when using dark mode

## [0.1.1] - 2026-03-13

### Added

- Dark mode with theme customization functionality

### Changed

- Improved mobile experience

## [0.1.0] - 2026-03-13

### Added

- Rank levels
- Toggle to enable Rank color backgrounds for Hero cards
- `app/state.js` that loads before everything else
- [README](./README.md) screenshot

### Changed

- Pop-up and close button corner radius
- Lessened amount of color variables

## [0.0.1] - 2026-03-13

### Added

- Page/view switching
- Hero card loading
- Rank selection
- Sorting and filtering
- Basic styling
